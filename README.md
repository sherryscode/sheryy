# sheryy
大学作业<br>
基于Spring+SpringMVC+MyBatis的教务管理系统<br>
项目启动需要用到java8,mysql(版本影响应该不大我的是8.0.43),Tomcat9.0.43<br>
项目启动后访问localhost:51234进入系统页面<br>
项目Clone地址：https://github.com/sherryscode/sheryy.git<br>
